# rsconnect integration


Manage OAuth integrations on Posit Connect.


``` bash
rsconnect integration COMMAND [ARGS]...
```


<span class="gd-details-chevron" aria-hidden="true"></span>Full --help output


    Usage: rsconnect integration [OPTIONS] COMMAND [ARGS]...

      Manage OAuth integrations on Posit Connect.

    Options:
      --help  Show this message and exit.

    Commands:
      add        Create a new OAuth integration.
      edit       Update an existing OAuth integration.
      list       List OAuth integrations.
      remove     Delete an OAuth integration.
      show       Show a single OAuth integration.
      templates  List and inspect OAuth integration templates.
