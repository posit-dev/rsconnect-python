# rsconnect info


Display information about a deployment. For any given file, information about its deployments are saved on a per-server basis.


``` bash
rsconnect info FILE
```


<span class="gd-details-chevron" aria-hidden="true"></span>Full --help output


    Usage: rsconnect info [OPTIONS] FILE

      Display information about a deployment. For any given file, information
      about its deployments are saved on a per-server basis.

    Options:
      --help  Show this message and exit.


# Arguments


`FILE: PATH`  
Required.
