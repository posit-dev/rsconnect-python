# rsconnect environment


Manage execution environments on Posit Connect.


``` bash
rsconnect environment COMMAND [ARGS]...
```


<span class="gd-details-chevron" aria-hidden="true"></span>Full --help output


    Usage: rsconnect environment [OPTIONS] COMMAND [ARGS]...

      Manage execution environments on Posit Connect.

    Options:
      --help  Show this message and exit.

    Commands:
      add     Create a new execution environment.
      edit    Update an existing execution environment.
      list    List execution environments.
      remove  Delete an execution environment.
      show    Show a single execution environment.
