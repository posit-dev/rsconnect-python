# rsconnect integration templates


List and inspect OAuth integration templates.


``` bash
rsconnect integration templates COMMAND [ARGS]...
```


<span class="gd-details-chevron" aria-hidden="true"></span>Full --help output

    Options:
      --help  Show this message and exit.

    Commands:
      list  List available OAuth templates.
      show  Show details of an OAuth template.
