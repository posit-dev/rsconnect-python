# rsconnect content repository


Manage git repository configuration for content items.


``` bash
rsconnect content repository COMMAND [ARGS]...
```


<span class="gd-details-chevron" aria-hidden="true"></span>Full --help output


    Usage: rsconnect content repository [OPTIONS] COMMAND [ARGS]...

      Manage git repository configuration for content items.

    Options:
      --help  Show this message and exit.

    Commands:
      delete    Remove git repository configuration from a content item.
      redeploy  Trigger a new deployment from the git repository.
      show      Show git repository configuration for a content item.


# Commands


`show`  
[Show git repository configuration for a content item.](../../../reference/cli/content/repository/show.md)

`delete`  
[Remove git repository configuration from a content item.](../../../reference/cli/content/repository/delete.md)
